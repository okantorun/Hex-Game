1)During the game,to return to the menu, you must write to the terminal "BACK MENU".
2)The program is case sensitive.
3)If "LAST MOVE" is written to the terminal during the game, the last move  will be printed.
4)"SAVE filename.txt" must be written to the terminal during the game for saving.
5)"LOAD filename.txt" must be written to the terminal during the game for loading.
